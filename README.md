# 8-Game Profile Lab

## Como rodar

1. Instale dependências:
   pip install -r requirements.txt

2. Rode:
   streamlit run app.py

## O que tem no app

- Dashboard geral
- Página de modalidades da biawhite
- Página de perfis de oponentes
- Página de comparação entre a biawhite e outros jogadores
- Análise local heurística
- Campo para análise com IA pela OpenAI, se você colocar sua API key
